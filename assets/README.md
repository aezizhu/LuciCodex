# Assets Directory

This directory contains project assets including the logo.

## Logo

The main project logo should be placed as `logo.png` in this directory.

**Requirements:**
- Format: PNG
- Recommended size: 480x480px or larger (will be displayed at 240px width)
- Transparent background preferred

The logo will automatically appear in:
- All README files (English, Chinese, Spanish)
- GitHub repository page (when configured)
